Descripció del llibre

Tot i que els inicis d'Internet es remunten als anys seixanta, no ha estat fins als noranta que, gràcies a la Web, el seu ús s'ha escampat per tot el món. En pocs anys, la Web ha evolucionat com mai: hem passat de pàgines senzilles amb poques imatges i continguts estàtics que visitaven uns quants usuaris a pàgines complexes i dinàmiques, amb continguts que provenen de bases de dades i que són visitades per milers d'usuaris alhora. Com uns centres comercials a l'hora punta, però digitals.

Totes les pàgines estan construïdes amb la mateixa tecnologia: el Llenguatge de marques d'hipertext, HTML, i els Fulls d'estil en cascada, CSS. Així és com es fan màgiques!

Aquest llibre és adequat per a qualsevol persona amb ganes de crear les seves pròpies pàgines web. No calen coneixements previs, només saber utilitzar un ordinador i navegar per la Web sense perdre's en massa finestres obertes.

  • Contingut del llibre

    A la primera part es treballen conceptes generals per desenvolupar pàgines web; s'expliquen les estructures física i lògica d'un lloc web. També es detalla com afecta l'estructura física a les URL i enllaços del lloc web. Passem pel concepte d'"estàndard web", un terme general per als estàndards que defineixen com funciona tot, com HTML i CSS. Aquest és el punt on s'entén que una URL no és només una sopa de lletres!

  • HTML

    A la segona part es treballa HTML. Partint de l'estructura bàsica d'una pàgina web, s'expliquen les etiquetes HTML per definir el text, enllaços, llistes, taules, formularis i elements multimèdia. La part divertida on els vídeos de gats i les taules amb emojis comencen a fer sentit.

  • CSS

    A la tercera i última part s'explica CSS, el llenguatge que fa que una web passi de semblar un treball d'institut a una web professional. S'explica com utilitzar els colors, com definir la presentació del text, de les taules i dels formularis, i com fer transformacions i transicions perquè el teu lloc web tingui un aspecte increïble. Finalment, la teva web semblarà un lloc al qual vols tornar, no només un experiment!
